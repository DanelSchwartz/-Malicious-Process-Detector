# Malicious Process Detector

This tool is part of an examination exercise designed to scan CSV log files for activities related to common attack tools, potentially indicating unauthorized or malicious activities within a network.

## Usage

The script analyzes a CSV file containing system logs. Replace `file_path` with the path to your CSV file containing the logs before running the script.

```bash
python extract_malicious_processes_details.py
```

## Background
This script was created as part of a cybersecurity examination to aid in the detection of potential security incidents through log analysis.
